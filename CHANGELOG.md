# 更新日志

## v1.4.0 (2026-05-09)

### 新增
- **双向删除策略** — Teable ↔ Teable 双向任务支持基于软删除标记的删除传播，避免把单侧缺失误判为删除
- **附件/图片字段同步** — 支持 Teable 原生附件数组透传，以及 SQL 文本/JSON URL 转 Teable 附件字段
- **生产日志配置** — 新增 `LOG_LEVEL`，服务端日志统一输出并脱敏敏感字段
- **大表压力测试脚本** — 新增 `npm run stress:e2e`，输出 1 千/1 万/10 万行压力测试 Markdown 报告

### 改进
- 前端主要面板异步加载，Vite 构建拆分 Vue、Element Plus、图标和 axios chunk
- README 补充双向删除、附件支持范围、日志级别和压力测试说明

### 限制说明
- 原始二进制 BLOB/Buffer/image 字段暂不直接上传到 Teable，会在预检和日志中提示跳过

## v1.3.0 (2026-05-08)

### 新增
- **局域网多人使用增强** — 用户管理增加搜索、角色筛选和更稳定的列表布局，连接共享继续保持敏感字段脱敏
- **Teable -> Teable 同步** — 支持 Teable 表到 Teable 表同步
- **双向同步和冲突策略** — 支持 Teable 双向同步、源端优先、目标端优先、最近更新时间优先和跳过冲突
- **任务模板和复制** — 可把已有同步任务保存为模板，也可复制任务快速创建相似配置
- **字段变更自动感知** — 支持字段快照、字段漂移检测和字段快照刷新
- **完整任务详情页** — 以概览、运行历史、高级信息和近期日志组织任务状态、配置、字段映射和失败批次
- **观测告警** — 以异常处理队列展示待处理告警、异常任务、24 小时成功率和告警通知状态
- **Webhook 告警通知** — 管理员可配置 Webhook URL、告警阈值、冷却时间和测试发送；Webhook URL 加密保存并脱敏返回
- **配置导入导出** — 支持环境迁移，默认导出移除连接密钥和告警 Webhook
- **同步前预检查** — 任务运行/启动前检查连接状态、映射、写入限制和兼容性

### 改进
- README 重写为面向部署和使用的文档，补充局域网部署、Docker 一键部署、数据源到告警推送的完整流程
- Docker 环境变量补充 `ALERT_NOTIFICATION_SCAN_INTERVAL_MS`
- 一致性校验按业务日期和时区归一化日期字段
- Teable 写入批量限制和前端提示保持一致

### 修复
- 修复同步任务启动提示和实际调度状态不一致的问题
- 修复连接测试状态在数据源列表中反复显示“未测试”的问题
- 修复 OAuth fallback 引用已删除变量的问题
- 修复共享连接列表泄露密钥字段的问题
- OAuth 登录回调改为短期 code 交换，避免 JWT 出现在 URL query 中
- 清理测试账号并迁移历史测试数据所有权，避免孤儿连接和任务

## v1.2.0 (2026-04-28)

### 新增
- **Teable OAuth 单点登录** — 登录页面新增「用 Teable 账号登录」按钮，支持通过 Teable OAuth 授权自动创建账号并登录
- **OAuth 回调自动处理** — 前端自动检测 URL 中的 `oauth_code` 参数，交换登录令牌后清理 URL
- **用户管理 API** — 超级管理员可查看/删除用户
- **密码修改** — 个人中心支持修改密码

### 修复
- OAuth token 端点从 `/api/oauth/token` 修正为 `/api/oauth/access_token`
- OAuth 请求 Content-Type 改为 `application/x-www-form-urlencoded`

## v1.1.0 (2026-04-28)

### 新增
- **用户认证系统** — JWT 登录/注册、bcrypt 密码加密
- **Teable OAuth 2.0 授权连接** — 在 Web UI 内完成 OAuth 授权流程，无需手动复制粘贴 Token
- **OAuth 状态管理** — 显示已连接账号、断开重连

### 改进
- README 新增 OAuth FAQ 和 API 接口文档

## v1.0.0 (2026-04-27)

### 新增
- **核心同步引擎** — SQL Server / MySQL / PostgreSQL → Teable 数据同步
- **全量同步 + 增量同步** — 基于时间戳列（updated_at）
- **Upsert / Skip 冲突策略** — 主键冲突自动更新或跳过
- **自动建字段** — 检测源表新增列，自动在 Teable 创建对应字段
- **字段类型映射** — SQL 类型自动转换为 Teable 类型
- **可视化 Web 管理界面** — 暗色主题仪表盘（连接管理 / 同步任务 / 日志面板）
- **实时同步日志** — WebSocket 推送
- **定时调度** — 手动触发或定时自动同步
- **连接测试** — 保存前验证连通性
- **Docker 部署支持** — Dockerfile + docker-compose.yml
